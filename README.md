## About Me

I like building AI wrappers that do more than wrap, crypto products with actual taste, and developer tools that feel fast enough to stay in your flow.

Current obsessions:

- AI agents, desktop app surfaces, chat UX, and code-native workflows.
- Solana, Ethereum, and especially the strange, quirky, high-surface-area L2 universe.
- Rust for systems, Go for services, TypeScript for product velocity.
- Small tools that turn scattered context into something shippable.
